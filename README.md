# Robotic-course
Robotic class repository, include each sessions codes and circuits.
-------------------------------------------------------------------

we use:
1) Arduino UNO
2) Gearbox *2
3) Battery (9V)
4) SRF05 (detect distance)
5) MQ2 or MQ5 (detect CO gas)
6) Jumper wire
7) Driver L298N
8) Bluetooth module
